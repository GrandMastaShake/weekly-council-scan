# History Archive

Closed position archives from the Paper Portfolio Tracker.

Each `.yaml` file contains the full record for one week:
- Entry prices for all 5 picks
- Exit prices (Friday close or stop loss)
- P&L per position and weighted portfolio return
- SPY benchmark return and alpha

Files are named by the Monday date of each week: `YYYY-MM-DD.yaml`

> *Do not edit these files manually. They are generated by `portfolio/tracker.py`.*
